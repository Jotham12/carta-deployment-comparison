Here we provide a guide how to use this scripts for Kubernetes for this deployment:  setup-mongodb.sh install mongodb, setup-microceph-microk8s.sh install microK8s and microceph, create-carta-config-secret.sh create carta-config.json secret, create-extrausers.sh add users that will be able to login in carta frontend, create-uploader.sh we just deploy busybox so that we can add fits images to our cephfs.
